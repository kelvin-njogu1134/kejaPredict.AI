KejaPredict.AI
 AI-Powered House Price Prediction System for Kenya

KejaPredict.AI is a machine learning-powered web application that predicts house prices based on key property features such as location, size, number of rooms, and income level.

It is designed to support tenants, landlords, buyers, sellers, and real estate investors in making smarter and data-driven decisions in the Kenyan property market.
